JavaScript

DOM Manipulation, Events, and Simple State

This page includes three JavaScript demos: a to‑do list, basic form validation, and a functional calculator. The scripts use vanilla JS (no frameworks) and emphasize accessibility. JavaScript is what makes websites interactive - it's like adding a brain to your HTML and CSS, allowing users to click buttons, fill out forms, and see real-time updates without refreshing the page.

To‑Do List

A simple task manager that demonstrates how JavaScript can create, read, update, and delete items from a list. This shows basic DOM manipulation - creating new elements, adding event listeners, and managing application state.

    Form Validation

    This form uses our global validation logic to check user input in real-time. It demonstrates how JavaScript can provide immediate feedback to users, making forms more user-friendly and reducing errors before submission.

    Calculator

    A fully functional calculator that demonstrates more complex JavaScript concepts like event handling, mathematical operations, and state management. Try performing some calculations to see how JavaScript handles user input and displays results.
